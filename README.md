# gitmetrics
CLI Tool for counting line of codes by file extension.
Example:
```
$ gitmetrics repo
Elapsed 2.003027888s
       ext       sloc
       ---        ---
     .java       720k
     .json       203k
     .wsdl        32k
      .jar        26k
                  24k
      .xml        23k
      .ttf        18k
       .vm        17k
       .js        13k
      .pdf         8k
      .svg         7k
     .yaml         5k
     .html         5k
       .gz         3k
     .lock         3k
```
